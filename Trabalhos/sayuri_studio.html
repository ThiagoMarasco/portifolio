Visão Geral

Desenvolvi um website sofisticado para um estúdio de tatuagem, com ênfase especial no design elegante e na funcionalidade intuitiva. O ponto alto deste projeto é a integração de um sistema de agendamento otimizado, projetado para proporcionar uma experiência de usuário fluida e sem complicações. Através desta plataforma, os clientes podem facilmente marcar seus horários, enquanto um recurso inovador permite a visualização e confirmação de agendamentos diretamente pelo WhatsApp.


O website foi estrategicamente dividido em duas versões distintas: uma destinada aos usuários e outra aos funcionários, assegurando que cada grupo tenha acesso apenas às funcionalidades relevantes para suas necessidades. A integração com o WhatsApp, realizada através do Twilio, centraliza o controle da agenda, permitindo o cancelamento e a confirmação de agendamentos de maneira eficiente e diretamente pela plataforma.

Desafios

A integração foi realizada utilizando a API do Twilio para WhatsApp. Implementei webhooks para capturar eventos de agendamento e disparar mensagens automáticas, como confirmações e lembretes de agendamento, diretamente do sistema para o WhatsApp dos clientes.


Implementei um sistema robusto de autenticação e autorização utilizando Django. Isso incluiu o uso de grupos de usuários e permissões específicas para separar claramente o acesso às funcionalidades do site, com base no papel de cada usuário.


A escolha da AWS EC2 como plataforma de hospedagem permitiu aproveitar os benefícios da escalabilidade. Configurei instâncias EC2 para escalarem automaticamente, ajustando-se à demanda e garantindo um desempenho estável. Além disso, otimizei as configurações do servidor Django e o banco de dados MySQL para eficiência.
